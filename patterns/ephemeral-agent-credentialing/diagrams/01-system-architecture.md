This diagram shows how all 8 core components of the Ephemeral Agent Credentialing pattern interconnect. Components are color-coded by category:

- **Purple** — Orchestration Layer
- **Indigo** — Identity & Credentials (Components 1-2)
- **Blue** — Zero-Trust Enforcement (Components 3-4)
- **Teal** — Multi-Agent Security (Components 6-7)
- **Amber** — Observability & Audit (Components 5, 8)
- **Pink** — Agent Instances
- **Gray** — Protected Resources
